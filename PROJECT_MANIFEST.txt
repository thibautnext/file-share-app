🚀 QUICK START
═══════════════════════════════════════════════════════════════

Step 1: Install
  cd /Users/thibaut/clawd/file-share-app
  npm install

Step 2: Configure Database
  - Create database on NAS: CREATE DATABASE file_share
  - Create folder: /volume1/docker/file-share/uploads
  - Copy .env.example to .env and fill database URL

Step 3: Migrate
  npm run migrate

Step 4: Run
  npm run dev

Step 5: Test
  Open http://localhost:3000
  Upload a test file
  Verify download works

Step 6: Deploy
  - Push to GitHub
  - Connect to Vercel
  - Configure environment variables
  - Run migrations in production
